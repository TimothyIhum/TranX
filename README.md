# TranX Vault

TranX is a privacy-first secure vault for storing sensitive information. It uses Supabase as the backend database and performs client-side encryption before data is uploaded. No sensitive vault content is stored in browser local storage.

## Features

- Layered authentication
- Encrypted storage
- Intuitive search
- Password-protected destructive actions
- Strong recovery mechanisms

## Technology Stack

- Frontend: React/Next.js
- Backend: Supabase (PostgreSQL, Authentication)
- Encryption: AES-256-GCM, Argon2id
